vl-ui-multiselect demo

Meer voorbeelden en documentatie.

Multiselect met meerdere opties

      
        <select multiple is="vl-multiselect">
          <option value="België">België</option>
          <option value="Duitsland">Duitsland</option>
          <option value="Frankrijk">Frankrijk</option>
        </select>

Multiselect met een voorgeselecteerde optie

      
        <select multiple multiple is="vl-multiselect" data-vl-multiselect>
          <option selected value="Brugge">Brugge</option>
          <option value="Brussel">Brussel</option>
          <option value="Gent">Gent</option>
        </select>

Multiselect listen to change events (watch console logs)

      
        <select multiple is="vl-multiselect" data-vl-multiselect id="change-listener">
          <option value="België">België</option>
          <option value="Duitsland">Duitsland</option>
          <option value="Frankrijk">Frankrijk</option>
        </select>
        <script type="text/javascript">
          document.getElementById('change-listener').addEventListener('change', (e) => {
            console.dir(`e.detail.value => ${e.detail.value}`);
            console.dir(`e.target.value => ${e.target.value}`);
          });
        </script>

Multiselect methoden

      <script type="text/javascript">
        <select id="vl-multiselect-methoden" multiple is="vl-multiselect" id="vl-multiselect-methoden">
          <option value="België">België</option>
          <option value="Duitsland">Duitsland</option>
          <option value="Frankrijk">Frankrijk</option>
          <option value="Nederland">Nederland</option>
          <option value="Italië">Italië</option>
        </select>
    
        <button is="vl-button" onclick="document.getElementById('vl-multiselect-methoden').dress()" type="button">Dress</button>
        <button is="vl-button" onclick="document.getElementById('vl-multiselect-methoden').undress()" type="button">Undress</button>
        <button is="vl-button" onclick="document.getElementById('vl-multiselect-methoden').enable()" type="button">Enable</button>
        <button is="vl-button" onclick="document.getElementById('vl-multiselect-methoden').disable()" type="button">Disable</button>
        <button is="vl-button" onclick="document.getElementById('vl-multiselect-methoden').setValueByChoice('Duitsland')" type="button">Kies Duitsland</button>
        <button is="vl-button" onclick="document.getElementById('vl-multiselect-methoden').removeActive()" type="button">Verwijder selectie</button>
        <button is="vl-button" onclick="document.getElementById('vl-multiselect-methoden').showDropdown()" type="button">Show dropdown</button>
        <button is="vl-button" onclick="document.getElementById('vl-multiselect-methoden').hideDropdown()" type="button">Hide dropdown
